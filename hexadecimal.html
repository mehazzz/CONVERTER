METHODS FOR CONVERSIONS

Binary to Hexadecimal conversion

To convert a binary number to its hexadecimal equuivalent:

  1. Start with the given binary number.
  2. Group the binary digits into sets of four digits from the right.
  3. If necessary, pad the leftmost group with zeros to make it a set of four digits.
  4. Convert each group of four binary digits to its hexadecimal equivalent.
  5. Combine the hexadecimal equivalents obtained from each group to get the final hexadecimal number.

Example: Binary number 1011101101

0010 1110 1101

In this case, padding is required for the leftmost group:

0010 becomes 0001 0010.
0001 in binary is 1 in hexadecimal.
0010 in binary is 2 in hexadecimal.
1110 in binary is E in hexadecimal.
1101 in binary is D in hexadecimal.

Hexadecimal number: 12ED

Octal to Hexadecimal conversion

To convert a octal number to its hexadecimal equuivalent:

  1. Start with the given octal number.
  2. Convert the octal number to its binary equivalent.
  3. Group the binary digits into sets of four from the right.
  4. If necessary, pad the leftmost group with zeros to make it a set of four digits.
  5. Convert each group of four binary digits to its hexadecimal equivalent.
  6. Combine the hexadecimal equivalents obtained from each group to get the final hexadecimal number.

Example: Octal number 753

octal equivalent: 111 101 011

grouping: 0011 1101 0110

In this case, padding is required for the leftmost group:

0011 becomes 0000 0011
0000 in binary is 0 in hexadecimal.
0011 in binary is 3 in hexadecimal.
1101 in binary is D in hexadecimal.
0110 in binary is 6 in hexadecimal.


Hexadecimal number: 36D6
